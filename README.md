# 3D Files and VR Assets

A curated collection of 3D models, scenes, and virtual reality assets for K-12 educational use.

## Contents

- **3D Models** — Printable and interactive 3D objects across subjects (science, history, math)
- **VR Scenes** — Immersive environments designed for classroom VR headsets
- **Textures & Materials** — Reusable surface assets for 3D scenes
- **AR Markers** — Augmented reality trigger images and linked assets

## Supported Formats

| Format | Use Case |
|--------|----------|
| `.stl` | 3D printing |
| `.obj` / `.fbx` | 3D modeling software |
| `.glb` / `.gltf` | Web-based and VR/AR viewers |
| `.usdz` | Apple AR Quick Look |

## Getting Started

1. Browse the folders by subject or grade level.
2. Download the asset in your preferred format.
3. Import into your VR platform (e.g., CoSpaces, Meta Horizon Worlds, Unity, Blender).

## Usage

These assets are intended for **educational, non-commercial use** by K-12 students and educators.

## Contributing
