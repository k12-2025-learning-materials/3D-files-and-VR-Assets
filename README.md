## Supported Formats

| Format | Use Case |
|--------|----------|
| `.stl` | 3D printing · GitHub 3D viewer |
| `.obj` | 3D modeling software (Blender, Maya) |
| `.fbx` | Game engines (Unity, Unreal) |
| `.glb` | Web-based and VR/AR viewers (Three.js, WebXR) |
| `.mp4` | 3D and immersive video playback |

## Getting Started

1. Click any **View 3D** link above to preview STL models directly in GitHub.
2. Download the asset in your preferred format.
3. Import into your VR platform (CoSpaces, Meta Horizon Worlds, Unity, Blender).

## Usage

These assets are intended for **educational, non-commercial use** by K-12 students and educators.
